GitHub Gist 연결

최초 1회만 설정하면, 이후 모든 기기에서
메모 목록이 자동으로 불러와집니다.

1. github.com → Settings → Developer settings → Personal access tokens
2. gist 권한만 체크 후 토큰 생성 (ghp_...)
3. 아래에 붙여넣기 → 연결하기 (Gist ID는 자동 생성)

이 값은 브라우저에만 저장되며 외부로 전송되지 않습니다.

목록 화면 하단에서 Gist ID를 확인할 수 있습니다.
