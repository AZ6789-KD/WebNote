GitHub Personal Access Token 발급
1. GitHub.com → 우측 상단 프로필 → Settings
2. 좌측 하단 Developer settings 클릭
3. Personal access tokens → Tokens (classic)
4. Generate new token (classic) 클릭
5. gist 권한만 체크 후 토큰 생성 (ghp_... 형태)
6. 생성된 토큰을 복사 — 페이지를 떠나면 다시 볼 수 없습니다
연결 설정
1. 상단 MEMO 로고 클릭 → 연결 설정 모달 열림
2. 발급받은 토큰 붙여넣기
3. 처음 사용: Gist ID 칸을 비워두면 자동 생성됩니다
4. 연결하기 클릭 → 동기화 시작
다기기 연동
1. 첫 기기에서 연결 완료 후 MEMO 로고 클릭
2. 입력 필드의 마스터 Gist ID 복사
3. 다른 기기에서 동일한 토큰 + Gist ID 입력
4. 모든 폴더와 메모가 자동 동기화됩니다

2. gist 권한만 체크 후 토큰 생성 (ghp_...)
3. 처음 사용 시 Gist ID를 비워두면 자동 생성됩니다
