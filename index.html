GitHub Gist 연결

토큰 1개 + 마스터 Gist ID 1개로 모든 기기에서 자동 동기화됩니다.

1. GitHub → Settings → Developer settings → Personal access tokens
2. gist 권한만 체크 후 토큰 생성 (ghp_...)
3. 처음 사용 시 Gist ID를 비워두면 자동 생성됩니다
4. 다른 기기 연동: 아래에 마스터 Gist ID를 입력하세요

브라우저에만 저장되며 외부로 전송되지 않습니다.
